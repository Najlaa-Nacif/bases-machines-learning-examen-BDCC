# Bases de Machine Learning – Examen BDCC

Ce dépôt contient les solutions des exercices de **régression** et de **classification** réalisés dans le cadre du module **Bases de Machine Learning** du Master **BDCC**.

## Contenu du dépôt

* `devoir-regression.ipynb` : Notebook de régression avec TensorFlow et implémentation From Scratch.
* `devoir_classification.ipynb` : Notebook de classification avec TensorFlow et implémentation From Scratch.
* `diabetes.csv` : Jeu de données utilisé pour l'exercice de classification.
* `ML_NACIF Najlaa.pdf` : Rapport PDF contenant les réponses des exercices de régression et de classification.

## Exercice 1 : Régression avec TensorFlow

L'exercice porte sur la mise en œuvre d'un modèle de régression linéaire à l'aide de TensorFlow.

Les principales étapes réalisées sont :

* Génération d'un dataset synthétique ;
* Construction et entraînement d'un réseau de neurones ;
* Évaluation des performances à l'aide des métriques **MSE** et **R²** ;
* Comparaison avec une implémentation réalisée uniquement avec NumPy.

## Exercice 2 : Classification avec TensorFlow

L'exercice consiste à développer un modèle de classification binaire sur le jeu de données **Diabetes**.

Les étapes réalisées sont :

* Préparation des données ;
* Construction d'un réseau de neurones avec TensorFlow ;
* Évaluation des performances à l'aide des métriques **Accuracy**, **Recall** et **F1-score** ;
* Implémentation d'une régression logistique *From Scratch* avec NumPy ;
* Comparaison des résultats obtenus avec TensorFlow.
